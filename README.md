# EeasyBuild
Build and run your cpp projects easier on windows

# Explanation what it does
Instead of compiling your project with "cmake --build ." and then running it with "PROJECT_NAME.exe" just write build and then compile and automatically execute it

# How to install
1. Download the file and paste it into a folder that is added to your paths
2. Reopen cmd

# How to use
Write build in cmd while you are in your build directory and that's it
